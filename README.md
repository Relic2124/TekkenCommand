철권8 커맨드 생성기 프로젝트 구조

이동 커맨드는 u, d, f, b, ub, uf, db, df, n으로 총 9개가 있음
up, down, forward, back, neutral의 약자이며, 2개의 조합은 각각의 대각선을 의미
공격 커맨드는 1, 2, 3, 4가 있음
각각 left punch, right punch, left kick, right kick에 해당
특수 커맨드는 heat burst, rage art가 있음

기본 키 세팅은 wasd와 numpad 12345679
w-u, a-b, s-d, d-f
numpad 4-1, 5-2, 1-3, 2-4, 6-1+2, 3-3+4
특수 커맨드인 heat burst는 2+3 또는 numpad 7
특수 커맨드인 rage art는 df+1+2 또는 numpad 9

철권8에서의 입력 감각과 동일한 느낌을 주기 위해, 대각선 입력을 단일로 받기 위해, 홀드를 입력받기 위해 키 입력은 60프레임 단위로 받음


페이지에는 입력칸과 출력칸이 있는데, 입력칸에는 키 입력으로 커맨드 작성을, 출력칸에는 준비된 이미지(/src/Tekken Notation PNGs)를 사용
이미지는 다음과 같은 규칙으로 네이밍되어 있음
    0: 공격 커맨드 입력 안함(사용 안함)
    1, 2, 3, 4: 공격 커맨드
    1+2, 1+2+3 등: 동시 입력
    b/f/u/d, bhold/fhold/uhold/dhold: 이동 커맨드 단일 입력 및 홀드
    n: 중립

출력은 프레임 단위로 업데이트됨
입력칸에서 엔터를 누를 시 텍스트를 적을 수 있음
입력칸에서는 쌍따옴표로 텍스트를 표시
출력칸에서는 네모 안에 텍스트를 써놓은 이미지로 표시

출력칸의 이미지는 다운로드할 수 있음
